VobBw 计划

简介

这是一个造“世界”的计划，造一台计算机，开发它的完整软硬件。设计一门编程语言，开发其编译器。
主要工作

  • VobBw OS 操作系统开发
  • VobBw– 语言设计及其编译器开发
  • 重要软件的开发和移植
  • * VobBw 计算机硬件设计制造

这个计划是我一直以来的愿望，希望有一台完全属于自己的计算机。我的目标仅仅是希望 VobBw OS 可以在 VobBw 上跑起来；可以用 VobBw– 语言编写一个可以进行一些简单运算的程序；移植一些喜欢的软件(vim…)。
总之一切从简！ 它将是完全开源的，之后将维护在 GitHub 或 Coding 上。 这个计划会很难，如果大家也有兴趣可以一起来玩儿。
