DFS与BFS

发表于 2018-02-05 | 分类于 算法 | 阅读次数:
字数统计: 1,184

简介

DFS（深度优先搜索）和BFS（广度优先搜索）都是应用广泛的搜索算法，同属于图算法。两种算法实际上就是两种不同的搜索策略。

区别

  • DFS的访问顺序：1 -> 2 -> 4 -> 7 -> 8 -> 3 -> 5 -> 6
  • BFS的访问顺序：1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7 -> 8

DFS沿着一个分支一直递归到最后，而BFS从起点一层一层向外扩展。
DFS用栈来实现，BFS用队列来实现。

BFS实现

在此题中BFS算法是很占优势的，因为它不需要枚举所有情况，如果在某一层遇到终点那最小步数必然是当前层数，而DFS算法需要枚举所有情况来更新最小步数最后才能确定。特殊情况下如果起点和终点是邻居，BFS只需一层搜索就可以解决，而DFS仍然需要枚举所有结果。当然DFS可以进行剪枝优化但在此题中效率明显是不如BFS的。 如果将此题改为问通过迷宫的方法数，那么此题就不能用BFS解决，因为BFS求的只是最短路并没有枚举所有的路线。

此图中，如果题目问从1走到3有多少种方法。 如果是DFS它会找到两种路线：1 -> 3 和 1 -> 2 -> 3。 而BFS在第一层时找到1 -> 3这条路后就将3标记为已访问，之后所有点都已被访问就不存在第二层了也就不存在1 -> 2 -> 3这条路了。
